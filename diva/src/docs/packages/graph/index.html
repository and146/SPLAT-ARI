The Diva graph package

Welcome to the Diva graph package documentation tree. The Diva graph package implements a solid graph visualization foundation. At its heart is a Swing-style MVC data and notification model. Extensible and pluggable graph rendering and layout tools complete the picture. Please see the design document for a full description of the package's components.
