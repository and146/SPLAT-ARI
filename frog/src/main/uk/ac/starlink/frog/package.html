Main classes managing the FROG application.

The classes in this package manage the FROG user interface and underlying data model. Using classes in this and associated packages time series data can be inspected and analysed.

Associated classes:

uk.ac.starlink.frog
Main user interface classes
uk.ac.starlink.frog.ast
Interface into JNIAST and the uk.ac.starlink.ast.* classes
uk.ac.starlink.frog.data
Classes for handling the data model
uk.ac.starlink.frog.iface
User interface and data interface classes
uk.ac.starlink.frog.iface.images
Classes for handling images user in the interface
uk.ac.starlink.frog.iface.themes
Classes for handling look and feel and themes
uk.ac.starlink.frog.plot
Interface into DivaPlot and assocaited classes for plotting the data
uk.ac.starlink.frog.util
Misc utilitiy classes
