SpecDataFactory

This class creates and clones instances of SpecData that are appropriate to the data format of the given spectrum specification.

The specification is usual the file name plus extension and any qualifiers (such as HDS object path, or FITS extension number). This is parsed and identified by the InputNameParser class.

Methods

public SpecData get(String specspec)
Check the format of the incoming specification and create an instance of SpecData for it.
  • Parameter
    specspec
    the specification of the spectrum to be opened (i.e. file.fits, file.sdf, file.fits[2], file.more.ext_1 etc.).
  • Returns the SpecData object created from the given specification.
  • Exceptions

    • SplatException - thrown if specification does not specify a spectrum.

public SpecData getClone(SpecData source, String specspec)

Create an clone of an existing spectrum by transforming it into another implementation format. The destination format is decided using the usual rules on specification string.
  • Parameters
    source
    SpecData object to be cloned.
    specspec
    name of the resultant clone (defines implementation type).
  • Returns the cloned SpecData object.
  • Exceptions

    • SplatException - maybe thrown if there are problems creating the new SpecData object or the implementation.

public static SpecDataFactory getReference()

Return reference to the only allowed instance of this class.
  • Returns reference to only instance of this class.
