SplatBrowser

This is the main class for the SPLAT program. It creates the browser interface that displays and controls the global lists of the currently available spectra and plots.

Using the menus and controls of this interface, spectra can be opened, removed and copied into the global list. Plots can created, have spectra added and be closed.

Groups of selected spectra can have their display inspected and changed using the controls of the related #SplatSelectedProperties object.

There are also a series of global, rather than spectra, specific tools that are made available via a toolbar. These include an animator tool and tools for performing simple spectral arithmetic, plus more trivial options, like choosing the look and feel.

The actual display and interactive analysis of spectra takes place in the plots (see #PlotControlFrame, #PlotControl and #Plot).

Methods

public boolean addSpectrum(String name)
Add a new spectrum to the global list.
  • Parameter
    name
    the name (i.e. file specification) of the spectrum to add.
  • Returns true if spectrum is added, false otherwise.

public void animateSelectedSpectra()

Display a tool for selecting from the global list of spectra and then repeatable displaying the selected sequence. This is meant to allow the browsing of a series of spectra, or simulate blink comparison sequence.

public void copySelectedSpectra()

Make copies of all the selected spectra. These are memory copies.

public void deSelectAllPlots()

Deselect all the plots.

public void deSelectAllSpectra()

Deselect all the spectra.

public void displaySelectedSpectra()

Display all the spectra that are selected in the global list view. Each spectrum is displayed in a new plot.

public int displaySpectra(int id, String list)

Display a list of spectra given by their file names, in a plot specified by its identifier. The file names are assumed to be in a space separated list stored in a single String. If the plot doesn't exist then it is created and the identifier of that plot is returned (which will be different from the one requested).
  • Parameters
    id
    the plot identifier number.
    list
    the list of spectra file names (or disk specifications).
  • Returns the id of the plot that the spectra are displayed in, -1 if it fails.

public int displaySpectra(String list)

Display a list of spectra given by their file names. The file names are assumed to be in a space separated list stored in a single String.
  • Parameter
    list
    the list of spectra file names (or disk specifications).
  • Returns the identifier of the plot that the spectra are displayed in, -1 if it fails.

public PlotControlFrame displaySpectrum(SpecData spectrum)

Display a spectrum in a new plot.
  • Parameter
    spectrum
    The spectrum to display.
  • Returns the plot that the spectrum is displayed in.

public PlotControlFrame displaySpectrum(String spectrum)

Add and display a new spectrum in a new plot.
  • Parameter
    spectrum
    the name (i.e. file specification) of the spectrum to add and display.
  • Returns the plot that the spectrum is displayed in.

public int getSelectedPlots()

Get a list of all plots current selected in the plot table.
  • Returns list of indices of selected plot, otherwise null.

public int getSelectedSpectra()

Get a list of all spectra current selected in the global list.
  • Returns list of indices of selected spectra, otherwise null.

public void multiDisplaySelectedSpectra(boolean fit)

Display all the currently selected spectra in the currently selected plot, or create a new plot for them.
  • Parameter
    fit
    whether to make all spectra fit the width and height of the plot

public void removePlot(PlotControlFrame plot)

Remove a plot from the global list.
  • Parameter
    plot
    reference to the frame (i.e. window) containing the plot (these have a one to one relationship).

public void removeSelectedPlots()

Remove the currently selected plots from the global list and this interface.

public void removeSelectedSpectra()

Remove the currently selected spectra from the global list and this interface.

public void saveSpectrum(int globalIndex, String spectrum)

Save a spectrum to disk file.
  • Parameters
    globalIndex
    the index on the global list of the spectrum to save.
    spectrum
    the file to write the spectrum into.

public void selectAllPlots()

Select all the plots.

public void selectAllSpectra()

Select all the spectra.

public void showBinaryMathsWindow()

Show window for performing simple binary mathematical operations between spectra.

Show window for performing simple unary mathematical operations on a spectrum.
