This super( String scopename )

super( name ) returns a bsh This reference to the enclosing scope (method scope) of the specified name. e.g.: 
    foo() {
        x=1;
        bar() {
            x=2;
            gee() {
                x=3;
                print( x ); // 3
                print( super.x ); // 2
                print( super("foo").x ); // 1
            }
        }
    }
