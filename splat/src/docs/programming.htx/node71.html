Scripting Threads

BeanShell `This' type references also implement the standard Runnable interface, so you can declare a `run()' method in your bsh objects and make it the target of a Thread. 

    foo() {
        run() {
            // do work...
        }
        return this;
    }

    foo = foo();
    new Thread( foo ).start();
You can also use the bg() command to run an external script in a separate thread.
