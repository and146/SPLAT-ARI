MEMSpecDataImpl

This class provides an implementation of SpecDataImpl to access spectra stored in existing memory. All values are copied into arrays stored in memory.

The main use of this class is for temporary, generated and copied spectra.

Constructors

public MEMSpecDataImpl(String name)
Constructor - just take a symbolic name for the spectrum, no other significance.
  • Parameter
    name
    a symbolic name for the spectrum.

public MEMSpecDataImpl(String name, SpecData spectrum)

Constructor, clone from another spectrum.
  • Parameters
    name
    a symbolic name for the spectrum.
    spectrum
    a SpecData object to copy.

Methods

public long getAst()
Return reference to AST frameset that defines the coordinate relations used by this spectrum.
  • Returns reference to a raw AST frameset.

public double getData()

Return a copy of the spectrum data values.
  • Returns reference to the spectrum data values.

public double getDataErrors()

Return a copy of the spectrum data errors.
  • Returns reference to the spectrum data values.

public String getDataFormat()

Return the data format.
  • Returns the String "MEMORY".

public int getDims()

Return the data array dimensionality (always length of spectrum).
  • Returns integer array of size 1 returning the number of data values available.

public String getFullName()

Return the full name of spectrum. For memory spectra this has no real meaning (i.e. no disk file or URL) so always returns a string reminding users that they need to save it.
  • Returns the String "Memory spectrum".

public String getShortName()

Return a symbolic name.
  • Returns a symbolic name for the spectrum. Based on the filename.

public void save()

Save is just a copy for this class.
  • Exceptions

    • SplatException - never thrown for this implementation.

public void setData(double[] data, double[] coords)

Set the spectrum data. No errors.
  • Parameters
    data
    the spectrum data values.
    coords
    the spectrum coordinates, one per data value.

public void setData(double[] data, double[] coords, double[] errors)

Set the spectrum data. With errors.
  • Parameters
    data
    the spectrum data values.
    coords
    the spectrum coordinates, one per data value.
    errors
    the errors of the spectrum data values.
