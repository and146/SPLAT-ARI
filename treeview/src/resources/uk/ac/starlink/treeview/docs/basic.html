Treeview  

Treeview is a tool for displaying hierarchical structures, and knows about astronomical file formats amongst others. Operation is very intutive.

One panel shows the hierarchy as a tree of expandable nodes,

the other (this one) shows help or description of a selected node.

You can drag the separator between them or resize the whole window.

Tree panel controls:

  • Click a handle (/) to expand/collapse a node
  • Click node text to select it and show details
  • Right-click node text to reload or display as a different type

Detail panel controls:

  • Click tab names at the top to see different data views

Use the other tabs in this help window to find more information about Treeview. The Demo button () is also useful for exploring its capabilities.
